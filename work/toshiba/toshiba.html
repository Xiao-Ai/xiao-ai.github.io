Introduction

Toshiba Elevator is a system for collecting data, checking and tracking issues after the elevator is installed. Their previous version of Toshiba Elevator System was not user friendly at all. And because of the annoying bugs and issues inherently in the system, Toshiba Corporation eagerly need a better system to support their work.

There are two parts in the system. One is mobile part, which is for submitting data via mobile app when the operators are working onsite. The other one is the desktop part, which is for managers to check and examine the data.

What I did for Toshiba Elevator
1. Redesigned the tool for creating forms.

We created Smart Form, which is a form tool that allow users to customise forms without any coding. It is just like creating a questionnaire. All they need to do is to drag a question template from the “builder”, and drop it.

  • Intuitive drag and drop experience
  • 8 clicks V.S. few clicks all on one page
  • Require coding V.S. Build your own form in just a few minuets




2. Added Filter for Data Tables.
  • A clear and formatted data with just a glance is what users need.
  • Long and huge data table VS. Easy Filter with Key Words




3. Redesigned Everything.

We basically redesgined everything for Toshiba Elevator. Besides those core functions I mentioned above, the biggest difference is that we created a whole new style for it: more comfortable colors, more thoughtful layout, etc..
